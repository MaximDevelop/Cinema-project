Назначение редактора Figma - предназначен для индивидуальной или командной работы над графическими проектами.

Как скачать картинку из макета:
1. Открыть проект в Figma и выбрать нужный элемент или фрейм.
2. В правой панели свойств найти секцию Export и нажать на значок «+».
3. Выбрать желаемый формат: PNG, JPG, SVG или PDF.
4. Установить необходимое разрешение (1x, 2x или 3x) в зависимости от требований к качеству.
5. Нажать кнопку «Export» и выбрать папку для сохранения файла.

Назначение вкладки inspect -  предназначена для просмотра свойств элементов дизайна.

Шапка сайта — это верхний блок веб-страницы, который выполняет навигационные и брендовые функции.

Тег <header> в языке разметки HTML соответствует шапке сайта.

Для чего необходим файл normalaize.css? - необходим для стандартизации стилей по умолчанию в различных браузерах.

Вставьте ссылку для скачивания normalaize.css - necolas . github . io / normalize . css — файл normalize css

Пропишите команду для подключения normalaize.css - <link rel="stylesheet" href="path/to/normalize.css">.

Как подключить шрифт Roboto? - Шрифт Roboto можно подключить в веб-проект через сервис Google Fonts или локально.
